# MusicAutoFormatter Implementation Plan

이 문서는 ffmpeg와 Python을 사용하여 음악 파일의 포맷을 변환하고, 태그 및 앨범 아트를 자동화하는 프로그램의 개발 계획입니다.

## 1. 개요
현재 `flac_to_mp3.bat`을 통해 수동으로 수행하던 작업을 Python으로 자동화하며, Windows와 Linux에서 모두 실행 가능한 크로스 플랫폼 환경을 지원합니다.

### 주요 기능
- **포맷 변환**: `flac`, `wav`, `mp3` 파일을 고음질 `mp3` (320kbps)로 변환/재인코딩.
- **메타데이터(ID3 Tag) 관리**: 제목, 아티스트, 앨범 등 기본 태그 자동 입력.
- **앨범 아트 자동 최적화**: 
  - 내장 앨범 아트 추출 및 크기 검사.
  - 2MB 초과 시 800x800 해상도로 리사이징 및 압축 (1MB 내외 목표).
  - 결과 파일에 자동 삽입.

## 2. 기술 스택
- **언어**: Python 3.x
- **외부 종속성**: `ffmpeg` (시스템에 설치되어 있어야 함)
- **주요 라이브러리**:
  - `mutagen`: ID3 및 기타 오디오 메타데이터 처리.
  - `Pillow (PIL)`: 이미지 리사이징 및 최적화.
  - `subprocess`: ffmpeg 명령어 호출.

## 3. 구현 세부 계획

### Phase 1: 파일 스캔 및 변환 엔진
- 지정된 디렉토리 내의 `*.flac`, `*.wav`, `*.mp3` 파일을 탐색합니다.
- `ffmpeg`를 사용하여 오디오 데이터를 추출하고 `320k` mp3로 변환합니다.

### Phase 2: 메타데이터 처리 로직
- 원본 파일에서 메타데이터를 최대한 추출합니다. (특히 FLAC의 Vorbis Comment 등)
- 파일명을 기반으로 정보를 추측하거나 (필요 시), 전용 설정(Sidecar 파일 등)을 통해 태그를 입력합니다.
- ID3v2.3/2.4 표준을 사용하여 호환성을 확보합니다.

### Phase 3: 앨범 아트 프로세싱
- 이미지 데이터가 2MB를 초과하는지 체크합니다.
- 초과 시 `Pillow`를 사용하여 `800x800` 크기의 `PNG` 또는 고품질 `JPG`로 변환합니다.
- 변환된 이미지를 MP3 파일에 `APIC` 프레임으로 삽입합니다.

### Phase 4: 사용자 인터페이스 및 사용성
- CLI(Command Line Interface) 기반으로 구현합니다.
- `/input` 디렉토리와 `/output` 디렉토리를 구분하여 원본을 보존합니다.
- 진행 상황을 실시간으로 출력합니다.
